## Install and Uninstall Python3 Packages
**To Install**
```
pip3 install <package-name>
```
Or
```
sudo pip3 install <package-name>
```
**To Uninstall**
```
pip3 ununstall <package-name>
```
Or
```
sudo pip3 uninstall <package-name>
```
